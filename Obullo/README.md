### Installation

Create your composer.json file

```json
{
    "autoload": {
        "psr-4": {
            "": "app/classes"
        }
    },
    "require": {
        "obullo/obullo": "^master",
        "obullo/service": "^1.0",
        "ircmaxell/password-compat": "^1.0"
    }
}
```
